Instructions

This project is designed to test your understanding of conditional and looping statements. There are a total of five separate web pages and a "Navigation bar" to link between them:

  • This home page with instructions
  • A page that displays a term grade based upon input of a semester course average using a linear nested if ...else block
  • A pages that displays the day of the week based upon input of a day abbrevation or number using a switch block
  • A page that displays a list of celsius and fahrenheit temperatures inside an HTML table using a for loop
  • A page that displays the words in a sentence or phrase "split" into "tokens" using the split() gunction, an array and a for loop
